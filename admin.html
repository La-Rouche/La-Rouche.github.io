حفظ دائم على الموقع

عشان الحفظ يبقى دائم، اللوحة بتكتب الملف مباشرة على GitHub. محتاج توكن منك مرة واحدة — بيتخزن في المتصفح ده بس ومش بيتبعت لأي مكان تاني ولا بيترفع على الموقع.

  1. افتح صفحة إنشاء التوكن
  2. Resource owner: اختار La-Rouche
  3. Repository access: Only select repositories ← اختار La-Rouche.github.io بس
  4. Permissions ← Repository permissions ← Contents = Read and write (مفيش صلاحية تانية خالص)
  5. حدّد مدة انتهاء (مثلاً 90 يوم) ← Generate token ← انسخه والصقه تحت
⚠️ التوكن ده زي مفتاح — أي حد يفتح اللوحة من الجهاز ده هيقدر ينشر. متفتحش اللوحة على جهاز مش بتاعك، واستخدم «نسيان التوكن» لما تخلص.
